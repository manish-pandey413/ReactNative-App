** Get started
*** Clone the repository
#+BEGIN_SRC bash
git clone https://https://github.com/manish-pandey413/ReactNative-App
cd ReactNative-App
*** Install dependencies
   #+BEGIN_SRC bash
   npm install
   #+END_SRC
*** Start the app
   #+BEGIN_SRC bash
   npx expo start
   #+END_SRC
